a quick example

here is a look at a basic setup for hooking methods while maintaining proper closure status using generalized iteration, which is much cleaner for the modern environments i work in.

-- basic environment utility
local function secure_hook(target, callback)
    for key, value in target do
        if type(value) == "function" then
            target[key] = newcclosure(function(...)
                print("calling: " .. key)
                return callback(value, ...)
            end)
        end
    end
end

secure_hook(game_mt, function(original, ...)
    return original(...)
end)
